# dissertation
This repository contains the code, output logs and plots fundamental to my dissertation. 

Please note that the Roshambull dataset used was originally collected by the Red Bull Roshambull Facebook application and shared with researchers by the Roshambull development team, it was used in: Jaffe, S. & List, J.A. (2019). 'Behavior in Strategic Settings: Evidence from a Million Rock-Paper-Scissors Games.'
https://www.mdpi.com/2073-4336/10/2/18

The raw data is not included in this repository. The primary contact of the datset is Sonia Jaffe, a Principal Researcher Economist in Microsoft. She can be contacted on spj@uchicago.edu.
